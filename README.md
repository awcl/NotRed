# notred

Offline blue team reference tool. Single HTML file, no dependencies, no server required.

**298 KB** - [live version](https://awcl.github.io/notred/) or [download index.html](https://github.com/awcl/notred/blob/main/index.html) and open in any browser.

## What's in it

- **33 reference entries** across 6 domains: Security Fundamentals, Phishing Analysis, Threat Intelligence, Digital Forensics, SIEM, Incident Response
- **17 interactive calculators**: Email Header Decoder, URL Defang/Refang, Base64, Hash Identifier, Event ID Lookup, Subnet/CIDR, Epoch Converter, Port Lookup, HTTP Status Codes, IP Type Checker, TTL Identifier, Hex/Dec/Oct/Bin/ASCII, Byte Converter, Artifact Path Lookup, Cron Parser, ROT13/ROT47, Forensic Timestamps
- **62 curated tool links** across 5 categories with one-click OSINT pivots

Inline copy buttons on all code blocks, calculator outputs, and reference paths. Works fully offline.
